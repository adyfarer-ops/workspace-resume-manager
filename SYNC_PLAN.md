# 数据库联动方案

## 目标
实现本地 JSON 与 Supabase 数据库的双向联动

## 方案

### 方案 1: 单向同步（本地 → 数据库）
修改本地 JSON → 运行同步脚本 → 更新数据库 → 页面刷新显示新数据

### 方案 2: 双向同步（推荐）
- 本地修改 → 自动同步到数据库
- 数据库修改 → 本地页面实时更新
- 使用 Supabase Realtime 订阅

### 方案 3: 定时同步
- 设置定时任务，定期同步本地数据到数据库
- 适合不频繁更新的场景

## 实现步骤

1. 创建数据库表（如果不存在）
2. 实现同步脚本
3. 配置实时订阅
4. 测试联动效果
