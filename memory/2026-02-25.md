# Memory - 2026-02-25

## Resume PDF Generation System

### Created Files
1. **Dynamic Resume Template** (`resume-dynamic-template.html`)
   - Responsive flexbox layout (not fixed positioning)
   - Left sidebar (320px) + right content area (flexible)
   - Uses Tailwind CSS + custom styles
   - Print-friendly with @media print styles

2. **Data Filler Script** (`/tmp/generate_dynamic_resume.py`)
   - Reads from website JSON data files
   - Dynamically generates skills, experience, projects sections
   - Outputs filled HTML resume

3. **PDF Generator** (`generate-pdf.js`)
   - Uses Puppeteer with system Chrome
   - Generates A4 PDF from HTML
   - 2-page output with background colors

### Generated Outputs
- `resume-dynamic.html` - Filled HTML (290KB)
- `resume-dynamic.pdf` - PDF version (1.2MB, 2 pages)

### Key Features
- Dynamic layout adapts to content length
- Skills section auto-generated from data
- Work experience items dynamically created
- Projects rendered with different layouts per type
- Responsive design for different screen sizes

### Data Sources
- `/data/profile.json` - Personal info, contact, about
- `/data/experience.json` - Work history
- `/data/projects.json` - Project details
- `/data/skills.json` - Technical skills
- `/data/education.json` - Education background
- `/project/public/avatar.png` - Profile photo

### Usage
```bash
# Generate HTML
python3 /tmp/generate_dynamic_resume.py

# Generate PDF
node generate-pdf.js
```

### API Service Created (Port 3003)

**New File:** `resume-api.js` - Express API server

**Endpoints:**
| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/resume` | GET | View resume HTML in browser |
| `/api/resume/download/html` | GET | Download resume HTML file |
| `/api/resume/pdf` | GET | View resume PDF in browser |
| `/api/resume/download/pdf` | GET | Download resume PDF file |
| `/health` | GET | Health check endpoint |

**Features:**
- Real-time HTML generation from JSON data
- On-demand PDF generation with Puppeteer
- Proper Content-Disposition headers for downloads
- CORS enabled for cross-origin requests
- Auto-filename based on profile name

**Usage:**
```bash
# Start server
node resume-api.js

# Test endpoints
curl http://localhost:3003/health
curl http://localhost:3003/api/resume
curl -O http://localhost:3003/api/resume/download/pdf
```
